- **Tiago de Gregori Portfolio Website**

        - This portfolio website showcases few academic and personal software projects, including Java, JavaScript, Python, and React Native assessments. The site is built using HTML, modern CSS (Flexbox, Grid, animations), and fully responsive design. It includes project galleries, assessment summaries, documentation links, and external resources such as PDFs and GitHub repositories.
    
    - *Features*
        
        - Responsive multi-page layout built with Flexbox, Grid, and media queries.
        - Custom design using gradients, transitions, transforms, and animations (e.g., scroll arrow animation on the home section)
        - Project galleries with lightbox image previews for each assessment (Weather App, Monster Mayhem * 2, Java Zoo, LMS, Moodify).
        - Dedicated pages outlining assessment requirements, system features and grading results.
        - Clean navigation with a reusable header and footer across all pages
        - Integrated external links to GitHub repositories, social-media and downloadable CVs, Diplomas & Certificates.
    
    - *Page Structure*

        - index.html — Home page with introduction, social links, featured projects section and a footer.
        - weather-app.html — React Native Weather App overview, gallery, and assessment breakdown.
        - monster-mayhem-hex.html / monster-mayhem-backend.html — Game board and backend assessments.
        - library.html / dublin-zoo.html — Java-based assessments with galleries and requirement summaries.
        - moodify.html — Python-based weather/playlist automation project.

        - Each project page includes a gallery section, lightbox implementation, assessment requirements, and grading screenshots plus a header and a footer reused from index.html.
    
    - *CSS*

        - style.css — Core styling: layout, typography, gradients, animations (including the scroll-arrow pseudo-element).
        - projects.css — Styling for project-specific pages (galleries, lightboxes, requirements cards).
        - responsive.css — Breakpoints for 1024px, 768px, and 425px to adjust layout, images, columns, and spacing.
        - reset.css — Base reset for consistent rendering across browsers.

    - *Technologies Used*

        - HTML5
        - CSS3 (Flexbox, Grid, animations, transitions)
        - Google Fonts
        - Font Awesome icons
        - Responsive layout techniques
        - Lightbox using the CSS :target selector

- **Purpose**
        - This portfolio shows the work I completed in my software development modules. It brings all my assessments together in one place and displays each project with images, descriptions, and links to the full code.
